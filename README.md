# bookmakerXML

<!-- badges: start -->

<!-- badges: end -->

`bookmakerXML` is a Rstudio add-in for displaying odds from
bookmaker.eu.

`bookmakerXML` downloads odds from
[lines.bookmaker.eu](https://lines.bookmaker.eu) in xml format.

## Installation

Only on github

``` r
remotes::install_github("bookmakerXML")
```
